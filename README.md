# Path of Building MK2

A Rust port of [Path of Building Community](https://github.com/PathOfBuildingCommunity/PathOfBuilding).
Pure-Rust calc engine, egui desktop UI, eventual Wasm support.

## Status

**Phase 0 — Reconnaissance done.** See [docs/architecture-current.md](docs/architecture-current.md)
for the map of the source codebase being ported.

## Workspace layout

| Crate | Role |
|---|---|
| `crates/pob-engine` | Pure calc engine. No I/O, no UI, Wasm-clean. |
| `crates/pob-data` | Data types + loaders. Shared by engine and UI. |
| `crates/pob-extract` | Build-time tool: reads PoB's `src/Data/` Lua tables, emits Rust-loadable files. |
| `crates/pob-ui` | egui application. |
| `app/pob-desktop` | Thin binary wrapping `pob-ui` in eframe. |
| `data/` | Generated data files (output of `pob-extract`). |

## Building

```bash
cargo build --workspace
cargo run -p pob-extract --release  # one-time, populates data/
cargo test --workspace
```

`data/` is regenerated by `pob-extract` from the upstream PoB checkout and is
gitignored. Round-trip tests skip silently when it's missing.

### Continuous integration

[`.github/workflows/ci.yml`](.github/workflows/ci.yml) runs on every PR and push to
`main`:

- `cargo build --workspace --all-targets`
- `cargo test --workspace` (data-dependent integration tests self-skip)
- `cargo fmt --check` and `cargo clippy --workspace --all-targets` run as
  *advisory* jobs while the existing fmt/clippy drift is cleaned up — they do
  not block merge today.

The optional `pob_diff` regression job (`workflow_dispatch` only — see issue
[#7](https://github.com/jonatanferm/PathOfBuildingMK2/issues/7)) runs the
upstream PoB Lua harness against MK2's calc engine and prints the divergence
table. Trigger it from the Actions tab when you want to verify a PR against
the live PoB calc; the upstream PoB repo is checked out at job time.

### Running pob_diff locally

```bash
# One-time setup: clone PoB upstream alongside the Rust port.
git clone https://github.com/PathOfBuildingCommunity/PathOfBuilding .PathOfBuilding

# Diff against any of the bundled fixtures.
cargo run --release -p pob-extract --bin pob_diff -- \
  --build crates/pob-extract/test-builds/witch_l90_arc_with_items.xml
cargo run --release -p pob-extract --bin pob_diff -- \
  --build crates/pob-extract/test-builds/marauder_l90_bleeding_cleave.xml
```

Output prints a side-by-side comparison of the 26 curated probe keys plus an
auto-discovered list of every shared scalar key. See
[`docs/divergences.md`](docs/divergences.md) for the divergence breakdown and
known follow-ups.

## Reference repo

The upstream Lua codebase lives in `.PathOfBuilding/` (a hidden subdir, gitignored)
inside this repo. The `pob-extract` tool reads from there. If you have an older
clone with the upstream sitting at the sibling path `../PathOfBuilding/`, that
still works — `pob-extract` and `pob_diff` try `.PathOfBuilding/` first and fall
back to `../PathOfBuilding/`. Or pass `--pob PATH` to override either default.

To set up the new layout from scratch:

```sh
git clone https://github.com/PathOfBuildingCommunity/PathOfBuilding .PathOfBuilding
```
